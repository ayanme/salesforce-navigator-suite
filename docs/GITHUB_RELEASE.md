# GitHub Release Reference

This document contains recommended GitHub metadata for the public release of the Salesforce Navigator Suite. It is intended for the repository owner and does not form part of the user-facing documentation.

---

## Repository Description

A suite of Claude Skills for researching Salesforce using official documentation.

---

## Suggested Topics

```
claude
claude-skills
salesforce
salesforce-ai
agentforce
marketing-cloud
data-cloud
developer-tools
prompt-engineering
documentation
```

Topics can be added under **Repository → Settings → Topics** on GitHub.

---

## Suggested Repository Settings

| Setting | Value |
|---|---|
| Visibility | Public |
| License | MIT (already set via LICENSE file) |
| Issues | Enabled |
| Discussions | Optional — enable if community Q&A is desired |
| Wiki | Disabled (documentation lives in `docs/`) |
| Releases | Enabled — tag `v1.0.0` against the initial commit |

---

## First Release

Create a GitHub Release tagged `v1.0.0` with the title **Salesforce Navigator Suite — v1.0.0**.

The release notes can be drawn from `CHANGELOG.md [1.0.0]`.
